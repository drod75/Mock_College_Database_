## Getting Started
To get a local copy of Mock_College_Database up and running locally follow these steps:  

### Prerequisites
1. Make sure you have MySQL installed.

### Setup
1. Navigate to the directory where you want to clone/run/save the application:
    ```sh
    cd example_directory
    ```
2. Clone the repository:
    ```sh
    git clone https://github.com/drod75/Mock_College_Database_.git
    ```
3. Navigate to the project directory:
    ```sh
    cd Mock_College_Database
    ```
